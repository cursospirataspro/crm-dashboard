Paso 1
Instalar el plugin

          

            

              
1

              

                Sube la carpeta del plugin a WordPress
                
Copia la carpeta wordpress-plugin/cpp-dashboard-crm-api dentro de wp-content/plugins/ de tu servidor (por FTP, cPanel File Manager o SSH).

              

            

            

              
2

              

                Activa el plugin
                
Ve a WordPress → Plugins y activa CPP CRM Dashboard API for WooCommerce.

              

            

            

              
3

              

                Define el token en wp-config.php
                
Abre wp-config.php (raíz del servidor) y añade antes de /* That's all */:

                
define('CPP_CRM_DASHBOARD_TOKEN', 'CAMBIA_ESTO_POR_ALGO_LARGO');

                
Usa mínimo 32 caracteres aleatorios. Este token es la contraseña del API.

              

            

            

              
4

              

                Verifica que el endpoint responde
                
Abre en tu navegador:

                
https://cursospirataspro.com/wp-json/cpp-crm-dashboard/v1/overview

                
Debe devolver {"code":"forbidden"...} — eso confirma que el plugin está activo y el endpoint existe.
